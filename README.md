# GeoDig - 交互式地质年代探索与化石挖掘模拟器

一个基于 React + TypeScript + Canvas + D3.js 的交互式地质探索应用。

## 功能特性

### 🎮 核心玩法
- **地质剖面挖掘**：Canvas渲染纵向地质剖面（地表→前寒武纪，12个年代层），程序化生成岩层纹理
- **三种挖掘工具**：
  - ⚙️ **钻头**：挖掘速度快，但可能损坏化石
  - 🔨 **地质锤**：平衡的挖掘工具，中等速度和精度
  - 🖌️ **刷子**：精细清理，100%保存化石完整度

### 🦴 化石系统
- 预置 **55+ 种化石矿物**，涵盖动物、植物、矿物三大类
- 每层按概率随机埋藏，SVG轮廓半透明显示在岩层中
- 挖掘到化石时弹出知识卡片，包含详细描述和趣味知识
- 化石完整度系统：不同工具影响化石完整度

### 📅 D3.js地质时间轴
- 垂直时间轴标注12个地质年代的名称、时间、生物事件
- 与地质剖面精确对应，点击可高亮对应年代
- 支持"地球时间压缩"动画，快速浏览地球历史

### 🏛️ 收藏馆 (/collection)
- 展示所有挖掘成果
- 支持按年代/类型/稀有度筛选
- 显示每种化石的完整度百分比
- 未发现的化石显示为问号

### 💾 数据存储
- 使用 IndexedDB 本地存储
- 支持多存档管理
- 自动保存游戏进度

### 🏆 成就系统
- 8个成就徽章：
  - ⛏️ 初露锋芒：完成第一次挖掘
  - 🖌️ 刷子爱好者：使用刷子挖掘100次
  - 🦖 恐龙猎人：发现5种不同的恐龙化石
  - 🏆 化石收藏家：收集20种不同的化石
  - 🌋 远古探索者：探索前寒武纪地层
  - 📚 古生物学家：收集5种古生代化石
  - ✨ 完美发掘：发现一个完整度100%的化石
  - ⏰ 时间旅行者：在所有12个地质年代都有发现

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：TailwindCSS 3
- **数据可视化**：D3.js v7
- **本地存储**：IndexedDB (idb)
- **路由**：React Router v6

## 游戏玩法

1. **创建存档**：首次进入游戏时，输入存档名称开始探索
2. **选择工具**：在右侧工具栏选择合适的挖掘工具
3. **挖掘地层**：在Canvas上点击并拖动鼠标来挖掘地层
4. **发现化石**：挖掘到化石时会弹出知识卡片，自动加入收藏
5. **探索时间轴**：点击左侧时间轴的年代可以高亮显示
6. **查看收藏**：导航到收藏馆页面查看所有已发现的化石
7. **解锁成就**：完成特定目标解锁成就徽章
8. **时间压缩**：点击"时间压缩"按钮观看地球历史动画
